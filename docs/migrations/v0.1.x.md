# Migration guide — v0.1.x (initial release)

This is the first tagged release of golusoris. No breaking changes from
pre-alpha main — this guide documents the stable API surface established at v0.1.0.

## Core changes from pre-alpha

### `config.Config` is now always passed as `*config.Config`

fx params structs must use `*config.Config` (pointer), not `config.Config` (value).
The value form triggers a govet `copylocks` error because `Config` embeds a mutex.

**Before:**
```go
type params struct {
    fx.In
    Config config.Config
}
```

**After:**
```go
type params struct {
    fx.In
    Config *config.Config
}
```

### `clock.Clock` replaces all `time.Now()` calls

All time operations in framework code use the injected `clockwork.Clock`.
In fx-wired code, accept `clock.Clock` via `fx.In`. In tests, use `clockwork.NewFakeClock()`.

**Before:**
```go
ts := time.Now()
```

**After:**
```go
// in production:
ts := clk.Now()  // clk clockwork.Clock injected via fx

// in tests:
fake := clockwork.NewFakeClock()
fake.Advance(5 * time.Second)
```

### Errors must wrap with `fmt.Errorf("pkg: action: %w", err)`

All errors must be wrapped following the `pkg: action: upstream` convention.
Bare `err` returns (without wrapping) cause `wrapcheck` lint failures.
