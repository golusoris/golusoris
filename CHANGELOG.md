  `gerr.Code`), `clock` (clockwork), `id` (UUIDv7 + KSUID), `validate`,
  `i18n`, `crypto`.

  `testutil/pg` testcontainers harness.

  cv,3d,game}`, `ocr`, `pdf` + `pdf/parse`, `markdown`, `hash`, `fs/watch`.

### Security

- CSRF: migrated from gorilla/csrf to [filippo.io/csrf/gorilla](https://pkg.go.dev/filippo.io/csrf/gorilla)
  (same-origin enforcement, matches Go 1.25 `net/http.CrossOriginProtection`).
  Closes GHSA-82ff-hg59-8x73; no upstream patch available in gorilla/csrf.
- All CI workflow `uses:` pins to commit SHAs; gosec pinned to v2.25.0.
- CodeQL advanced setup with `security-extended` + `security-and-quality`
  query packs.
- pgx `GO-2026-4771`/`GO-2026-4772` affect `pgproto3.Backend` (server-side
  wire protocol); govulncheck confirms 0 reachable in framework code. No
  upstream fix as of release date.
