## [0.1.0] — 2026-04-14

First tagged release. Framework is usable via
`go get github.com/golusoris/golusoris@v0.1.0`.

### Added

- **Core** — `config` (koanf), `log` (slog + tint), `errors` (go-faster +
  `gerr.Code`), `clock` (clockwork), `id` (UUIDv7 + KSUID), `validate`,
  `i18n`, `crypto`.
- **DB** — `db/pgx` pool, `db/migrate` (golang-migrate), `db/sqlc` helpers,
  `testutil/pg` testcontainers harness.
- **HTTPX** — `httpx/server`, `httpx/middleware`, `httpx/router` (chi),
  `httpx/client`, plus `cors`, `csrf` (filippo.io/csrf/gorilla), `ratelimit`,
  `ws`, `form`, `static`, `htmx`, `geofence`, `autotls`, `vite`,
  `rangeserve`. OpenAPI server generation via `ogenkit` + ogen; Scalar API
  docs via `apidocs`.
- **Observability** — `otel` (tracer/meter/logs + OTLP), `observability/sentry`,
  `observability/profiling` (Pyroscope), `observability/pprof`,
  `observability/statuspage`.
- **Runtime** — `k8s/{podinfo,health,metrics,client}` with workload identity;
  `container/runtime`, `leader/{k8s,pg}` pluggable election; `systemd`
  sd_notify + watchdog; Prometheus scrape helpers in `tools/`.
- **Jobs + outbox** — `jobs` (river), `jobs/{cron,ui,workflow}`, `outbox`,
  `outbox/cdc`, `testutil/river`.
- **Cache** — `cache/{memory,redis,singleflight}` (otter + rueidis),
  `testutil/redis`.
- **Auth + authz** — `auth/{oidc,passkeys,jwt,apikey,magiclink,linking,
  impersonate,session,recovery,policy,lockout,captcha,oauth2server,scim}`,
  `authz` (casbin).
- **Notify + realtime** — 11 notify providers (Resend, Postmark, SendGrid,
  Mailgun, AWS SES, Twilio, FCM, APNs, Web Push, Telegram, Teams) behind a
  unified `notify.Sender`; `realtime/{sse,pubsub,webrtc}`.
- **Webhooks** — `webhooks/{in,out}`.
- **SaaS primitives** — `tenancy`, `idempotency`, `flags` (OpenFeature),
  `audit`, `page` (cursor + offset pagination).
- **Files + media** — `storage` (gocloud.dev), `archive`, `media/{av,img,
  cv,3d,game}`, `ocr`, `pdf` + `pdf/parse`, `markdown`, `hash`, `fs/watch`.
- **Search + AI** — `search/{meilisearch,pgfts,typesense}`,
  `ai/{llm,vector,tiny}` (Anthropic + OpenAI + Ollama + LiteRT helpers).
- **Commerce** — `payments/{stripe,subs,meter,invoice}`, `money`.
- **Integrations** — `geoip`, `secrets` (Vault + cloud SM + K8s ESO).
- **Big stacks (opt-in, own `go.mod`)** — `grpc`, `graphql` (gqlgen +
  genqlient), `db/{clickhouse,timescale,geo}`, `pubsub/{kafka,nats}`,
  `ebpf`, specialty modules (`science/{plot,bio,numerical}`, `hw/{gpio,
  robotics,fssnap,udev}`, `media/{game,3d}`, `web3/{evm,solana}`,
  `net/{smtpserver,dnsserver,wol}`, `docs/{docx,xlsx,epub}`,
  `testutil/pact`).
- **Misc** — `clikit` (cobra), `clikit/tui` (bubbletea), `selfupdate`,
  `plugin`.
- **Testing** — `testutil/{snapshot,factory,fuzz,load,mutation,prop,fxtest}`.
- **CLI + MCP** — `cmd/golusoris` (init / add / bump), `cmd/golusoris-mcp`.
- **Deploy** — `deploy/{helm,observability,logging,terraform,flux,argocd,
  crossplane}`.
- **AI layer** — `.claude/{skills,hooks}`, `docs/upstream/`,
  `docs/migrations/`, per-package `AGENTS.md` + `CLAUDE.md`.
- **CI + supply chain** — 7 GitHub workflows (CI, reusable ci-go, CodeQL
  advanced setup, OpenSSF Scorecard, reusable release-go, rebuild-on-base,
  auto-assign). goreleaser + cosign keyless + syft SBOM + SLSA-L3
  provenance on tagged releases.

### Security

- CSRF: migrated from gorilla/csrf to [filippo.io/csrf/gorilla](https://pkg.go.dev/filippo.io/csrf/gorilla)
  (same-origin enforcement, matches Go 1.25 `net/http.CrossOriginProtection`).
  Closes GHSA-82ff-hg59-8x73; no upstream patch available in gorilla/csrf.
- All CI workflow `uses:` pins to commit SHAs; gosec pinned to v2.25.0.
- CodeQL advanced setup with `security-extended` + `security-and-quality`
  query packs.
- pgx `GO-2026-4771`/`GO-2026-4772` affect `pgproto3.Backend` (server-side
  wire protocol); govulncheck confirms 0 reachable in framework code. No
  upstream fix as of release date.

### Known limitations

- `ai/tiny/serve/tflite` Python sidecar client deferred to v0.2.0.
- `integrations/goenvoy` adapter deferred to v0.2.0 (upstream library at
  `github.com/golusoris/goenvoy` itself ships in v0.1.0).

### Migration notes

First release — no prior version to migrate from. Subsequent releases
will accumulate `Migration:` footers from commits with breaking changes;
`golusoris bump <version>` applies codemods automatically.
